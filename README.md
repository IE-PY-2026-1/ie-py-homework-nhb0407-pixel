## 2. 시나리오 (5~10줄)
최근 한정판 RPG 굿즈나 피규어 시장은 주식 시장만큼이나 가격 변동이 심합니다. 초보 수집가들은 적정 가격을 몰라 '바가지'를 쓰거나, 판매 타이밍을 놓쳐 손해를 보기도 합니다.
본 프로그램은 이러한 문제를 해결하기 위해 '데이터 기반의 스마트 수집 보조 도구'를 지향합니다. 사용자가 관심 있는 굿즈의 정보(정가, 현재 시세, 매물 상태 등)를 입력하면, 시스템은 축적된 데이터를 바탕으로 현재 시장의 과열 상태를 진단합니다.
특히, 본 프로그램의 핵심인 'AI 자동매매 시뮬레이션' 모드는 사용자가 설정한 '매수 목표가'와 '매도 목표가'를 기준으로 작동합니다. 시장 가격 데이터가 업데이트될 때마다 프로그램은 실시간으로 로직을 가동하여, 저평가된 매물이 나오면 자동으로 '가상 장바구니'에 담고(매수), 목표 수익률에 도달하면 자동으로 '판매 완료' 처리를 하여 자산 수익률을 극대화하는 과정을 시뮬레이션합니다. 이를 통해 사용자는 복잡한 계산 없이도 자신의 수집 포트폴리오를 효율적으로 관리하고, 최적의 매매 타이밍을 연습해볼 수 있습니다.

# 🚀 [버전별 개발 일지 & AI 협업 기록]
## 🟦 [1차 과제: V1.0] 시나리오 기획
    
# 개발 내용
변수 활용: goods_name (문자열), purchase_price (정수), profit_rate (실수) 등 다양한 자료형을 정의하고 활용했습니다.
형변환(Casting): 파이썬의 input()은 모든 입력을 문자로 취급하기 때문에, 계산을 위해 int()를 사용하여 숫자로 바꾸어 주는 핵심 문법을 포함했습니다.
f-string 포맷팅: f"{variable:,}"를 사용하여 금액에 콤마(,)를 넣었습니다.
##코드 
# =========================================================
# [Resell-Master V1.0] 굿즈 시세 분석 프로그램
# 작성자: (남현빈)
# 과제 목표: 변수 선언, 입출력, 산술 연산을 이용한 프로그램 구축
# =========================================================
# 1. 인트로 화면 출력
print("=========================================")
print("     Goods Market Analyzer V1.0")
print("      나의 굿즈 수집 포트폴리오")
print("=========================================")
print()
# 2. 데이터 입력 (input 사용)
# input()은 기본적으로 문자열로 받으므로, 숫자는 int()로 형변환이 필요합니다.
goods_name = input("1. 분석할 굿즈 이름: ")
purchase_price = int(input("2. 개당 구매 가격(원): "))
market_price = int(input("3. 현재 장터 시세(원): "))
quantity = int(input("4. 보유 수량(개): "))
# 3. 데이터 연산 (에너지/로직 처리)
# 변수를 활용하여 필요한 수치들을 계산합니다.
total_cost = purchase_price * quantity      # 총 투자 금액
total_value = market_price * quantity       # 현재 총 가치
net_profit = total_value - total_cost       # 예상 순수익
profit_rate = (net_profit / total_cost) * 100  # 수익률(%)
# 4. 최종 결과 리포트 출력
print()
print("-----------------------------------------")
print(f"[ 분석 결과: {goods_name} ]")
print(f"- 총 투자 금액: {total_cost:,}원")
print(f"- 현재 총 가치: {total_value:,}원")
print("-----------------------------------------")
print(f"▶ 예상 순수익: {net_profit:,}원")
print(f"▶ 현재 수익률: {profit_rate:.2f}%")
print("-----------------------------------------")
print("※ V2.0에서 조건문(if)을 이용한 등급 판정 기능이 추가될 예정입니다.")
print("=========================================")

    * **적용 내용:** 단순 입출력을 넘어, **'기초 정보 입력 및 리스트 저장(1차) → 전투 등급 판독기 구현(2차) → 무한 메뉴 시스템 구축(3차) → 딕셔너리를 활용한 다중 캐릭터 관리(4차)'**로 이어지는 논리적인 확장 로드맵을 확립함.
    
### **🛠️ Troubleshooting & 기술 회고:**
(예시) 
1. **문제 1: 숫자를 입력했는데 계산이 안 되는 현상 (TypeError)**
    * **원인:** `input()`으로 받은 데이터는 기본적으로 '글자'라서 숫자와 곱셈 연산이 불가능했음.
    * **해결:** `int()`와 `float()` 함수를 사용하여 입력값을 숫자로 바꾸는 '형변환' 마법을 부려 해결함.
2. **문제 2: 조건문 뒤에 자꾸 생기는 빨간 줄 (SyntaxError)**
    * **원인:** `if`와 `elif` 조건식 끝에 마법의 기호인 `:`(콜론)을 자꾸 빠뜨려서 발생함.
    * **해결:** 파이썬의 모든 조건문과 반복문 끝에는 항상 땡땡(`:`)이 들어가야 한다는 규칙을 학습하고 수정함.
